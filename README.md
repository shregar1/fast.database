# fastmvc-db-models

Shared SQLAlchemy ORM models intended to be reused across multiple FastMVC-based apps and domains.

This package provides:
- A shared `Base` declarative instance
- Common ORM models under `fastmvc_db_models.models.*`
- Shared DB table name constants under `fastmvc_db_models.constants.*`

## Development

This repo is set up to be installed locally via:

`pip install -e fastmvc/fastmvc_db_models`
